Get AUTOUI running in 3 steps.

## Step 1: Install

```bash
npm install @autoai-ui/autoui
```

## Step 2: Basic Config

Create your config file:

**React:**
```tsx
// lib/autoui-config.ts
import { AutoUIConfig } from "@autoai-ui/autoui"

export function createAutoUIConfig(): AutoUIConfig {
  return {
    appId: 'my-app',
    llm: {
      proxyUrl: process.env.REACT_APP_PROXY_URL || 'http://localhost:3001',
      sharedSecret: process.env.REACT_APP_SHARED_SECRET,
    },
    runtime: {
      validateLLMOutput: true,
      storeChatToLocalStorage: true,
    },
  }
}
```

**Next.js:**
```tsx
// lib/autoui-config.tsx
"use client"

import { AutoUIConfig } from "@autoai-ui/autoui"

export function createAutoUIConfig(): AutoUIConfig {
  return {
    appId: 'my-app',
llm: {
      proxyUrl: process.env.NEXT_PUBLIC_PROXY_URL || 'http://localhost:3001',
      sharedSecret: process.env.NEXT_PUBLIC_SHARED_SECRET,
    },
    runtime: {
      validateLLMOutput: true,
      storeChatToLocalStorage: true,
    },
  }
}
```

## Step 3: Add ModalChat

**React:**
```tsx
import { ModalChat } from "@autoai-ui/autoui"
import { createAutoUIConfig } from "./lib/autoui-config"

function App() {
  const config = createAutoUIConfig()
  return <ModalChat config={config} />
}
```

**Next.js:**
```tsx
"use client"

import dynamic from "next/dynamic"
import { createAutoUIConfig } from "@/lib/autoui-config"

const ModalChat = dynamic(
  () => import("@autoai-ui/autoui").then(m => m.ModalChat),
  { ssr: false }  // ‚Üê Required for Next.js
)

export default function App() {
  const config = createAutoUIConfig()
  return <ModalChat config={config} />
}
```

## Environment Variables

Create `.env` (React) or `.env.local` (Next.js):

```bash
REACT_APP_PROXY_URL=http://localhost:3001
REACT_APP_SHARED_SECRET=your-secret
```

or

```bash
NEXT_PUBLIC_PROXY_URL=http://localhost:3001
NEXT_PUBLIC_SHARED_SECRET=your-secret
```

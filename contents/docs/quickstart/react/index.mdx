Get AUTOUI running in your React app in 5 minutes.

## Step 1: Install AUTOUI

```bash
npm install @autoai-ui/autoui
```

## Step 2: Set Up Backend Proxy

You'll need a backend proxy server. See [Backend Proxy Setup](/docs/backend-proxy) for details.

For now, you'll need:
- `PROXY_URL` - Your proxy server URL (e.g., `http://localhost:3001`)
- `SHARED_SECRET` - Your proxy shared secret

## Step 3: Create Your Config

Create `lib/autoui-config.ts`:

```tsx
import { AutoUIConfig } from "@autoai-ui/autoui"

export function createAutoUIConfig(): AutoUIConfig {
  return {
    appId: 'my-react-app',
    
    llm: {
      proxyUrl: process.env.REACT_APP_PROXY_URL || 'http://localhost:3001',
      sharedSecret: process.env.REACT_APP_SHARED_SECRET,
      appDescriptionPrompt: 'A React application with an AI assistant.',
    },
    
    runtime: {
      validateLLMOutput: true,
      storeChatToLocalStorage: true,
      maxSteps: 20,
    },
  }
}
```

## Step 4: Add ModalChat

In your main App component:

```tsx
import { ModalChat } from "@autoai-ui/autoui"
import { createAutoUIConfig } from "./lib/autoui-config"

function App() {
  const config = createAutoUIConfig()
  
  return (
    <div>
      <h1>My App</h1>
      <ModalChat config={config} />
    </div>
  )
}

export default App
```

## Step 5: Environment Variables

Create `.env` file:

```env
REACT_APP_PROXY_URL=http://localhost:3001
REACT_APP_SHARED_SECRET=your-shared-secret
```

## That's It!

You should now see a chat button in the bottom-right corner. Click it to start chatting!

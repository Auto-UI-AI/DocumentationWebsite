## What You'll Need

- A React or Next.js project
- A backend proxy server (we'll help you set this up)
- An OpenRouter API key (or similar LLM provider)

## Installation

```bash
npm install @autoai-ui/autoui
```

or

```bash
yarn add @autoai-ui/autoui
```

or

```bash
pnpm add @autoai-ui/autoui
```
